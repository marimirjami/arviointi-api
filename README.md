# Opiskelija- ja arviointijärjestelmä

Tämä on kurssin harjoitustyö, jossa on toteutettu Node.js (Express) -pohjainen API-rajapinta ja MySQL-tietokanta opiskelijoiden sekä arviointien hallintaan.

Ohjelman toiminta
- Sovellus mahdollistaa opiskelijoiden, kurssien ja arviointien hallinnan tietokannassa. API-rajapinta tukee seuraavia HTTP-metodeja:
* GET: Haetaan opiskelijoiden tietoja ja suorituksia tietokannasta.
* POST: Lisätään uusia opiskelijoita ja kirjautumistietoja.
* PUT: Päivitetään olemassa olevia opiskelijatietoja (esim. nimenmuutokset).
* DELETE: Poistetaan suorituksia järjestelmästä ID-numeron perusteella.
